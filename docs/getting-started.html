Getting Started

Choosing the right tools for the job.

Prerequisites

Before building modern extensions, ensure you have the following installed:

  • Node.js (v14+) (Optional): Required if using frameworks like Vue. Run `node -v` to check.
  • Google Chrome (or Brave/Edge): For testing the extension.
  • Any Text Editor: VS Code Recommended.

Architecture: Vanilla JS vs. Frameworks

Vanilla JS (The "Basic" Way)

Best for simple tools, like a context menu item or a color changer.

  • Pros: No build step, lightweight, instant changes.
  • Cons: Hard to manage state (like a list of network requests).
  • Example: checks our `basic/` folder.

Vue.js (The "Modern" Way)

Best for complex UIs, like our Analytics Dashboard.

  • Pros: Reactive data (UI updates automatically), components.
  • Cons: Requires a build step (`npm run build`).
  • Example: The main `web-analytics` project.

Hands On: Build it Yourself

Want to build the "Basic" extension from scratch? Follow these simple steps in your code editor (like VS Code):

Step 1: Create a new folder named my-extension.

Step 2: Inside that folder, create a file named manifest.json and paste this code:

{
  "manifest_version": 3,
  "name": "My First Extension",
  "version": "1.0",
  "action": {
    "default_popup": "popup.html"
  }
}

Step 3: Create a file named popup.html (The UI) and paste this code: 

<!DOCTYPE html>
<html>
<body style="width: 200px; text-align: center;">
  <h3>My First Extension</h3>
  <button id="clickMe">Click Me</button>
  <p id="message"></p>
  <script src="popup.js"></script>
</body>
</html>

Step 4: Create a file named popup.js (The Logic) to make the button work:

document.getElementById('clickMe').addEventListener('click', () => {
    if (document.getElementById('message').textContent === 'Hello (Created using Vanilla JS)!') {
        document.getElementById('message').textContent = 'Hello World!';
    } else {
        document.getElementById('message').textContent = 'Hello (Created using Vanilla JS)!';
    }
});

Step 5: That's it! You can now load this folder into Chrome. Should look like this:
